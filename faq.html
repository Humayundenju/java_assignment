Important FAQ

What is Document Object Model (Dom)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

How do you select an element From the DOM ?

six methods to select an element from the DOM. 1. getElementById – search element by element_id 2.getElementsByTagName – search element by tag name (e.g., span, div) 3.getElementsByClassName – search element by class name 4. getElementsByName – search element by name attribute 5. querySelector – returns the first element that matches the specified selector 6.querySelectorAll – returns elements that match the specified selector.

What is event Delegation in the context of the DOM,and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future.

How do you manipulate an elements attributes and style using the DOM ?

The classList object only provide methods to change the class of an element. If you want to change other attributes like id, href, or src, you can use the setAttribute() method. The setAttribute() method accepts two arguments: 1.The name of the attribute to set 2.The value of the attribute to set
